IndexSearcher.Search(Weight, Filter, Int32, Sort, Boolean) Method

Just like Search, but you choose whether or not the fields in the returned FieldDoc instances should be set by specifying fillFields.

NOTE: this does not compute scores by default. If you need scores, create a TopFieldCollector instance by calling Create and then pass that to Search.
