SortField Constructor

Creates a sort, possibly in reverse, by terms in the given field, parsed to numeric values using a custom Parser.

subclass an existing numeric parser, or field is null

SortField(
   string field,
   Parser parser,
   bool reverse
);

Parameters

field
Name of field to sort by. Must not be null.
parser
Instance of a Parser, which must subclass one of the existing numeric parsers from FieldCache. Sort type is inferred by testing which numeric parser the parser subclasses.
reverse
True if natural order should be reversed.
