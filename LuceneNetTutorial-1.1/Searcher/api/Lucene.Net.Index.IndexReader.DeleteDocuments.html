Lucene.Net

IndexReader.DeleteDocuments Method

Deletes all documents that have a given term indexed. This is useful if one uses a document field to hold a unique ID string for the document. Then to delete such a document, one merely constructs a term with the appropriate field and the unique ID string as its text and passes it to this method. See DeleteDocument for information about when this deletion will become effective.
