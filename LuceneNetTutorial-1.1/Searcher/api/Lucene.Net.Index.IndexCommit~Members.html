Public Instance Methods

Delete Delete this commit point. This only applies when using the commit point in the context of IndexWriter's IndexDeletionPolicy. Upon calling this, the writer is notified that this commit point should be deleted. Decision that a commit-point should be deleted is taken by the IndexDeletionPolicy in effect and therefore this should only be called by its OnInit(T) or OnCommit(T) methods.
Equals Two IndexCommits are equal if both their Directory and versions are equal.
