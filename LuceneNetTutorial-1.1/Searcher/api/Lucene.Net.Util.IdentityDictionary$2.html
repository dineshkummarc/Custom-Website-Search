IdentityDictionary(TKey,TValue) Class

A class that mimics Java's IdentityHashMap in that it determines object equality solely on ReferenceEquals rather than (possibly overloaded) object.Equals(). NOTE: Java's documentation on IdentityHashMap says that it also uses ReferenceEquals on it's Values as well. This class does not follow this behavior
