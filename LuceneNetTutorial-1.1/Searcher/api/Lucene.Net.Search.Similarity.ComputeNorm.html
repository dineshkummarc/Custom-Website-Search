Similarity.ComputeNorm Method

Compute the normalization value for a field, given the accumulated state of term processing for this field (see FieldInvertState).

Implementations should calculate a float value based on the field state and then return that value.

For backward compatibility this method by default calls LengthNorm passing Length as the second argument, and then multiplies this value by Boost.

WARNING: This API is new and experimental and may suddenly change.
