MultiPhraseQuery Class

MultiPhraseQuery is a generalized version of PhraseQuery, with an added method Add. To use this class, to search for the phrase "Microsoft app*" first use add(Term) on the term "Microsoft", then find all terms that have "app" as prefix using IndexReader.terms(Term), and use MultiPhraseQuery.add(Term[] terms) to add them to the query.
