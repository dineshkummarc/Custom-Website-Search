IndexesOfJust like indexOf(int) but searches for a number of terms at the same time. Returns an array that has the same size as the number of terms searched for, each slot containing the result of searching for that term number.
IndexOfReturn an index in the term numbers array returned from getTerms at which the term with the specified term appears. If this term does not appear in the array, return -1.
