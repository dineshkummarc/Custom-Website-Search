ParallelMultiSearcher.Search(Weight, Filter, Collector) Method

Lower-level search API.

Collect is called for every matching document.

Applications should only use this if they need all of the matching documents. The high-level search API (Search) is usually more efficient, as it skips non-high-scoring hits.

This method cannot be parallelized, because Collector supports no concurrent access.
