IndexSearcher.SetDefaultFieldSortScoring Method

By default, no scores are computed when sorting by field (using Search). You can change that, per IndexSearcher instance, by calling this method. Note that this will incur a CPU cost.

public virtual void SetDefaultFieldSortScoring(
   bool doTrackScores,
   bool doMaxScore
);

Parameters

doTrackScores
If true, then scores are returned for every matching document in TopFieldDocs.
doMaxScore
If true, then the max score for all matching docs is computed.
