IsIgnoringOffsets (inherited from TermVectorMapper) 
IsIgnoringPositions (inherited from TermVectorMapper) Indicate to Lucene that even if there are positions stored, this mapper is not interested in them and they can be skipped over. Derived classes should set this to true if they want to ignore positions. The default is false, meaning positions will be loaded if they are stored.
TermVectorEntrySet The TermVectorEntrySet. A SortedSet of TermVectorEntry objects. Sort is by the comparator passed into the constructor.
