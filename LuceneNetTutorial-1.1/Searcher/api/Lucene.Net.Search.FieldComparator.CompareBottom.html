FieldComparator.CompareBottom Method

Compare the bottom of the queue with doc. This will only invoked after setBottom has been called. This should return the same result as Compare } as if bottom were slot1 and the new document were slot 2.

For a search that hits many results, this method will be the hotspot (invoked by far the most frequently).
