Weight.Scorer Method

Returns a Scorer which scores documents in/out-of order according to scoreDocsInOrder.

NOTE: even if scoreDocsInOrder is false, it is recommended to check whether the returned Scorer indeed scores documents out of order (i.e., call GetScoresDocsOutOfOrder), as some Scorer implementations will always return documents in-order.
NOTE: null can be returned if no documents will be scored by this query.
public abstract Scorer Scorer(
   IndexReader reader,
   bool scoreDocsInOrder,
   bool topScorer
);

Parameters

reader
the IndexReader for which to return the Scorer.
scoreDocsInOrder
specifies whether in-order scoring of documents is required. Note that if set to false (i.e., out-of-order scoring is required), this method can return whatever scoring mode it supports, as every in-order scorer is also an out-of-order one. However, an out-of-order scorer may not support NextDoc and/or Advance, therefore it is recommended to request an in-order scorer if use of these methods is required.
topScorer
if true, Score will be called; if false, NextDoc and/or Advance will be called.

Return Value

a Scorer which scores documents in/out-of order.
