CharStream Class

CharStream adds CorrectOffset functionality over TextReader. All Tokenizers accept a CharStream instead of TextReader as input, which enables arbitrary character based filtering before tokenization. The CorrectOffset method fixed offsets to account for removal or insertion of characters, so that the offsets reported in the tokens match the character offsets of the original Reader.
